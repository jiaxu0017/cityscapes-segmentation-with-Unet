# Readme
The model was trained on cityscape-dataset with classes merged into 8 main classes.
The results have only 3 classes marked on them.
 - human (Red)
 - construction (Green)
 - vehicle (Blue)

## List of classes
  - void
  - flat
  - construction
  - object
  - nature
  - sky
  - human
  - vehicle
